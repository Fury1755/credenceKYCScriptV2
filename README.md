## credenceKYCScriptV2
A Playwright script that interacts with Credence Advisory Pte Ltd's SharePoint site via REST API calls. Automates the KYC (Know-Your-Client) process.
Currently WIP.

## What is KYC?
KYC (Know-Your-Client) is a background check conducted on clients to prevent financial crimes.
In the context of this script, it involves searching on Google and Baidu, then taking screenshots of the search page.
The screenshots are then uploaded to the appropriate SharePoint directories.
Progress is documented on a human readable Excel file.
